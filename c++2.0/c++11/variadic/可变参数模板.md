可变参数模板函数
可变参数模板是 C++ 11 中引入的一个新特性，它允许我们定义一个可以接受可变数目参数的模板函数或模板类。

在了解模板函数和模板类之前，我们需要先知道两个概念：

模板参数包：使用 typename/class ... Args 来指出 Args 是一个模板参数包，其中可能包含了零个或多个模板参数
函数参数包：使用 Args ... rest 来指出 rest 是一个函数参数包，其中可能包含了零个或多个函数参数
